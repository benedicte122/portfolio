Accueil | Retour aux Projets

Objectif du projet

Le but est de créer une base de données pour gérer des informations de manière structurée et apprendre à manipuler des tables et les relations entre elles. L’objectif est de comprendre l’organisation des données et les interactions entre elles.

Méthode prévue

La démarche prévue :

  • Identifier les entités et leurs relations (ex. livres, utilisateurs).
  • Créer les tables et les champs nécessaires.
  • Remplir la base avec des données fictives pour tester.
  • Interroger la base pour vérifier le bon fonctionnement.

Résultat attendu

Je compte acquérir une compréhension claire de la logique des bases de données relationnelles et de l’importance des relations entre tables pour structurer efficacement l’information.
